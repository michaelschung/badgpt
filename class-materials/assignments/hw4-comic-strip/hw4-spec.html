Overview

In this assignment, you will create a web page that displays a 4-panel comic strip, and allows you to interact with it through the use of buttons.

Requirements

Code

  • Upon downloading the starter code, you'll notice that there's a decent amount done for you already. Before doing anything else, link the provided HTML and CSS files together, and load the page.
  • Take time to examine the code carefully, and make sure you understand the structure and styling of the page.
  • Change the name of hw4-FirstnameLastname.html to use your name.
  • The HTML document title (the one that shows up in the browser tab) should be HW4: Lastname (substitute your last name)
  • From this point on, it is highly recommended that you complete the assignment in the following order of requirements:
    1. There are four folders inside the img folder, each containing a Calvin & Hobbes comic strip split into four panels. Choose your favorite strip and add those four images into your HTML.
    2. Each image should be 300 pixels tall; allow the width to scale itself.
    3. By default, the images should be positioned in a row in the center of the image box (the black part of the screen). Pause the video right at the start to see what it should look like.
    4. Create a JavaScript file inside the js subfolder, and link it into your HTML file.
    5. Edit the HTML button tags so that they trigger functions in the JS file when clicked.
    6. In the JS file, write the functions to be called by the HTML buttons.
      • The Padding, Border, and Margin buttons are all toggle buttons, which means that each time one is pressed, it flips its property back and forth between two states.
      • When the Padding button is pressed, it should toggle between no padding and an 8 pixel padding around each image panel.
      • When the Border button is pressed, it should toggle between no border and a 3 pixel, solid, limegreen border around each image panel.
      • When the Margin button is pressed, it should toggle between no margin and a 10 pixel margin around each image panel.
        • Tip: the code for padding/border/margin is all very similar, so choose one of those to focus on first, and then the other two will be almost exactly the same.
      • When an arrow button is pressed, the entire comic strip should be pushed in that direction, along with any padding/border/margin that is currently applied. If the middle button is pressed, the strip should return to the center of the image box.
        • Note that if the comic strip has already been pushed, for example, in the upward direction, then a push in the rightward direction would result in the strip being positioned in the upper-right corner of the image box. Thus, there should be 9 distinct positions possible to reach using the arrow keys.
    7. Finally, note that the cursor should turn into a pointer when hovering over any button.
