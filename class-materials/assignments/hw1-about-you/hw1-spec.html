Requirements

Code

  • Change the name of hw1-FirstnameLastname.html to use your name.
  • The HTML document title (the one that shows up in the browser tab) should be HW1: Lastname (substitute your last name)
  • Make sure to include all of the following components (feel free to go above and beyond if you want!):
    • A header at the top of the page
    • 4 sections, each with 2 or more informative sentences about your life
    • 2 different types of header tags
    • Thoughtful use of the <p> tag
    • Thoughtful use of bold and italicized text
    • 3 images; make sure that no single image takes up an unreasonable amount of space
    • 1 list (ordered or unordered)
    • 2 hyperlinks that automatically open as new tabs
